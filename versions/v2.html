Context gaps
Sensor readings without product context — a number without meaning
1 critical gap
Oven Station B — no SKU-to-temperature profile mapping
Oven B reports temperature readings continuously. Takorin flags anomalies when readings deviate from a rolling average. But there is no mapping between the active SKU and the expected temperature range for that SKU. A reading of 182°F is normal for Pepperoni Classic but a HACCP critical limit breach for the gluten-free flatbread line — and both run on Oven B. Without this context, the risk model cannot evaluate whether a temperature reading is safe or dangerous. It can only detect that it has changed.
Impact: Sensor A-7 anomaly alerts for Oven B are evaluated against a generic rolling average, not a product-specific safety threshold. This means both false positives (normal variance flagged as risk) and false negatives (actual threshold breaches not caught) are possible.
